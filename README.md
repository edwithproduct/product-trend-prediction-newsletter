# AI 如何改變 Figma？

一個探索人工智慧如何重塑設計工具未來的互動式網站。

## 📖 關於本專案

這是一個單頁滾動式敘事網站，深入分析 AI 對 Figma 及整個設計工具生態系統的影響。通過流暢的滾動體驗，引導讀者理解設計師和產品經理在 AI 時代的角色轉變。

## 🎯 主要論點

- Figma 在 AI 時代前的優勢與批評
- AI 帶來的軟體開發流程變革（10-20x 提速）
- Product Builder 時代的來臨
- Vibe Coding 工具對 Figma 的衝擊
- Figma 的未來定位：從精細設計工具到快速思考工具
- 設計版本控制的新可能性

## 🛠️ 技術棧

- **框架**: React 18 + Vite
- **樣式**: Tailwind CSS
- **動畫**: Framer Motion
- **語言**: 繁體中文

## 🚀 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建構生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 📦 部署

本專案已配置可直接部署到 Vercel：

1. 推送程式碼到 GitHub
2. 在 Vercel 導入此 GitHub 倉庫
3. Vercel 會自動偵測 Vite 配置並部署
